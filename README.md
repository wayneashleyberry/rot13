> A stupid rot13 implementation, written in Go.

### Installation

```sh
go get github.com/wayneashleyberry/rot13/cmd/rot13/...
```

### Usage

```
rot13 'Hello, World!'
Uryyb, Jbeyq!
```

```
rot13 --decode 'Uryyb, Jbeyq!'
Hello, World!
```
